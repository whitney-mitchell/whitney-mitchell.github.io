Who I am

I am an enthusiastic web developer driven by a passion for community, creative problem-solving, and a general love of improving things. Whether in the context of logic, or styling, it's the nitty gritty stuff that I love most.

I am currently finishing the full-stack web development bootcamp at Nashville Software School, and feeling eager to find a place I can continue learning and building cool stuff with wonderful people.

It took me many years to recognize that my passions and skill set would be perfectly suited to programming, but once I started to scratch the surface, learning some basic HTML, CSS, and JavaScript, I honestly felt like my world opened up. Challenging myself in ways I never had before brought me such joy and empowerment.

I love learning, and working with enthusiastic people, so I feel extremely fortunate to be entering a field in which continued learning and passion are such integral components. Additionally, I am so very grateful to have finally found the thing I love doing, the thing I get to be when I grow up!

Projects

Below are several projects including my front-end and back-end capstones, and some JavaScript and Python exercises. Please click around! Deployed projects are noted, and github repository links are provided otherwise.

  • Height to Human

    Demo: Best viewed in Chrome!

    Github repo

    NSS front-end capstone built using AngularJS, Angular Bootstrap, CSS3 animations, and a Firebase back-end. Answers the question, "How many Danny Devitos tall is the Batman Building?

  • MakeWhat?

    Currently in development

    Github repo

    NSS back-end capstone built using Python in the Django REST Framework, with an AngularJS, front-end. MakeWhat is a full-stack web app through which users may search for DIY project ideas based on the supplies they already have.

  • Dynamic Cards

    a simple JavaScript exercise

    Demo

    Github repo

    An exercise in dynamically created elements, and event listeners, using vanilla JavaScript. Additionally, a CSS study in repeating-linear-gradients. Users may enter text in a text field, and a card is generated displaying that text as well as a functioning delete button.

  • Mathmagician

    a simple Command Line app written in Python

    Github repo

    A back-end exercise teaching unit-testing and Python syntax. The app may be used to generate integer, fibonacci, prime, and number sequences. The user specifies which type of sequence and how many numbers should be generated.
